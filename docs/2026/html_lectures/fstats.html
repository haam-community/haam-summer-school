ADMIXTOOLS 2

We will use Admixtools2 (Maier and Patterson 2024), the R implementation of admixtools. You can find the documentation of the package here https://uqrmaie1.github.io/admixtools/

A more comprehensive definition of f-statistics theoretical background can be found in Patterson et al. 2012 and Peter 2016

DATASETS

During this practical, we will use 2 dataset, both are subsets of the Allen Ancient DNA Archive v66 (Mallick & Reich 2023).

  • The “modern” dataset contains primarily modern populations, with the addition of 4 archaic genomes (“Altai_Neanderthal”, “Vindija_Neanderthal”, “Denisova3”, “Denisova25”) and the Chimp reference genome.

  • The “ancient” dataset contains a number of Upper Paleolithic genomes, and 5 Yoruba (meant to be used as outgroup).

f2

In Admixtools2, f2 statistics are the foundation of all other analyses. Since all other f-stats can be obtained as a linear combination of f2 statistics, Admixtools2 pre-computes f2 stats for the efficient computation of all other statistics.

f2_from_geno() computes pairwise f2 statistics from EIGENSTRAT (.geno, .snp, .ind) or PLINK (.bed, .bim, .fam) files. To save time, we won’t compute them now, you will find the pre-computed f2 blocks between all pairs of populations in the folder /vol/volume/fstats .

f3 outgroup

f3 admixture

f3 statistic can also be used to test for admixture. f3(A,B,C) tests whether A is a mixture of a population related to B and one related to C.

f4 ratio

Using the ratio of two f4 statistics, we can estimate the admixture proportions of a 2-way admixture.

f4(f2,"French",  "Yoruba", "Vindija_Neanderthal", "Chimp")$est / f4(f2, "Altai_Neanderthal", "Yoruba", "Vindija_Neanderthal", "Chimp")$est
[1] 0.01905146

The twigstats package (Speidel et al. 2025 - https://leospeidel.github.io/twigstats/), has a function that implements f4 ratio

twigstats::f4_ratio(f2, popO = "Chimp", popX = "French", pop1 = "Yoruba", pop2 = "Altai_Neanderthal", popI = "Vindija_Neanderthal")
   popO                popI   pop1              pop2   popX        val
1 Chimp Vindija_Neanderthal Yoruba Altai_Neanderthal French 0.01905144
           se
1 0.002681484

# find out which populations show denisovan admixture #
f4 = f4(f2, "Yoruba", modern, "Denisova3", "Chimp")
pops = f4[which(f4$z <= -3), ]$pop2

# compute f4 ratios #
f4_ratio = c()
for (pop in pops) {
  f4_ratio = rbind(f4_ratio, twigstats::f4_ratio(f2, popO = "Chimp", popX = pop, popI = "Denisova3", pop1 = "Yoruba", pop2 = "Denisova25"))
}
f4_ratio = f4_ratio[order(f4_ratio$val, decreasing = T), ]

MDS based on outgroup-f3 statistics

To visualise the data, a dimensionality reduction method such as a principal component analysis (PCA) or MultiDimensional Scaling (MDS) is often useful. For aDNA a PCA is often computed on high quality modern genomes, and ancient genomes are then projected into that reference space. An alternative approach, well suited when modern-day variation does not capture well ancient variation, is to compute an MDS on the ancient genomes themselves, using 1-f3out to build a distance matrix between all samples.

# compute pairwise f2 #
# f2_ancient = admixtools::f2_from_geno("ancient", maxmiss = 1, verbose = F)

# you can save f2 blocks, so for future use you just need to load the file #
# save(f2_ancient, file = "f2_ancient.RData")
load("f2_ancient.RData")

# compute pairwise f3-outgroup ##
targets = colnames(f2_ancient)
targets = targets[which(targets != "Yoruba")]
f3_out = f3(f2_ancient, "Yoruba", targets, targets)
f3_out = f3_out[2:4]
f3_out %>% pivot_wider(id_cols = pop2, names_from = pop3, values_from = est) %>% column_to_rownames(var = "pop2")-> f3_matrix

# compute mds #
mds = as.data.frame(cmdscale(1-f3_matrix, k = 2))
mds$ID = rownames(mds)

# plot #
ggplot(mds, aes(x = V1, y = V2, label = ID)) +
  geom_point() +
  geom_text_repel() +
  labs(x = "Dim 1", y = "Dim 2") +
  theme_minimal()

  • twigstats has implemented a function to do the same thing, try to use it

    Hint: a poplabels file has 4 columns: ID, POP, GROUP, SEX. For this exercise the first three can be identical and SEX can be set to NA

# we need to create a poplabels file first #
poplabels = data.frame(ID = colnames(f2_ancient), POP = colnames(f2_ancient), GROUP = colnames(f2_ancient), SEX = NA)
write.table(poplabels, "ancient.poplabels", row.names = F, col.names = T, quote = F)

# compute mds #
mds = twigstats::calc_mds(f2_ancient, poplabels = "ancient.poplabels", outgroup = "Yoruba")
mds = mds[which(mds$method == "MDS"), ]

# plot #
ggplot(mds, aes(x = PC1, y = PC2, label = ID)) +
  geom_point() +
  geom_text_repel() +
  labs(x = "Dim 1", y = "Dim 2") +
  theme_minimal()

# select inds to analyse #
fam = setNames(read.delim("modern.fam", header = F, sep = " ")[1:2], c("POP", "ID"))
mypops = c("Sardinian", "French", "Basque", "Finnish", "English")
outgroup = "Yoruba"
fam = fam[which(fam$POP %in% c(mypops, outgroup)), ]
fam$POP2 = fam$POP
fam[which(fam$POP != outgroup), "POP2"] = fam[which(fam$POP != outgroup), "ID"]

# compute f2 on a subset of the individuals #
f2 = f2_from_geno("modern", inds = fam$ID, pops = fam$POP2, verbose = F)

# compute pairwise f3-outgroup ##
targets = fam[which(fam$POP2 != outgroup), "POP2"]
f3_out = f3(f2, outgroup, targets, targets)
f3_out = f3_out[2:4]
f3_out %>% pivot_wider(id_cols = pop2, names_from = pop3, values_from = est) %>% column_to_rownames(var = "pop2")-> f3_matrix

# compute mds #
mds = as.data.frame(cmdscale(1-f3_matrix, k = 2))
mds$ID = rownames(mds)
mds = merge(mds, fam, by = "ID")

# plot #
ggplot(mds, aes(x = V1, y = V2, label = ID, colour = POP)) +
  geom_point(size = 3) +
  labs(x = "Dim 1", y = "Dim 2") +
  theme_minimal()
